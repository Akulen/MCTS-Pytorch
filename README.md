# How to Run

## Step 1: Create a venv and download the requirements

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Step 1.5: Activate the venv

```bash
. venv/bin/activate
```

## Step 2: Run the example

```bash
python mcts.py
```
